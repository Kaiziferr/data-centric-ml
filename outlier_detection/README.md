# Outlier Detection

### Isolation Forest
**Isolation Forest** is a tree-based model specifically designed for anomaly detection. Its core principle is that anomalous observations are easier to isolate than normal ones. To achieve this, the algorithm builds multiple binary trees using random splits over the features, measuring how many partitions are required to isolate each data point.
